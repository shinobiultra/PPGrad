Detailed Description

template<int Dim, typename DT>
class PPGrad::Tensor< Dim, DT >

Abstract Tensor class for all Eigen accelerated math on tensors.

This class is a wrapper around Eigen::Matrix and Eigen::Tensor and is supposed to be immutable. Each operation instantiates a new Tensor object.s

Template Parameters
TThe type of the underlying data (Eigen::Matrix or Eigen::Tensor).
DTThe type of the scalar data (i.e., individual elements of data) - float or double expected.
